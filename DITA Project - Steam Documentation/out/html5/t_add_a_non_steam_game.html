Add a Non-Steam Game

Even if you own a game on a platform other than Steam, you can still add it to your Steam Library.

Games added to your Steam Library using this process can be launched from the Steam client, but will lack other funtionality available to native Steam titles.
  1. In the bottom left corner, click Add a Game,
  2. Click Add a Non-Steam Game. A list of programs installed on your computer will appear.
  3. Scroll through the list or use the search bar to find the game you'd like to add.
    1. If the game you're looking for is not on the list, Steam could not automatically detect it on your computer. Click Browse to manually search your computer's files for the program you're looking for. It will be added to the list once you select it.
  4. Click the check box next to the programs you'd like to add.
  5. Click Add Selected Programs.

Manage Collections

Collections let you organize games in a manner of your choosing.

  1. On the left panel, right-click a game you want in a collection.
  2. Hover over Add to and select + New collection.
  3. Type in a a name for the collection.
  4. Click Create Collection. A panel will open on the right displaying the new collection you've created with the game you chose. There will also be a new folder on the left with the collection name and game under it.
  5. Click games listed on the left and drag them into the collection folder to add more items to that collection.

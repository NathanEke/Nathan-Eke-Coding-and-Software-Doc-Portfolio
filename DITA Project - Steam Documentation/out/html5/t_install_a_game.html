Install a Game

Before you can play your new Steam game, you have to install it onto your computer.

Make sure at least one of your storage drives has sufficient storage space for the game you wish to download.
  1. In the left panel, select the game you want to install.
  2. Click Install.
  3. Check whether or not to create a desktop and start menu shortcut for the game.
  4. Select a storage drive with sufficient space.
  5. Click Install. How long it takes to download will depend on the size of the game and your internet speed. You will receive a notification on the bottom right of your screen once it is done.
